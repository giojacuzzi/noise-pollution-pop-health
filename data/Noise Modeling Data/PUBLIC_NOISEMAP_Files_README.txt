File Types:

*.baseops -- NOISEMAP or MRNMAP case file
*.xml -- NOISEMAP operations input file
*.poi -- results output
*.elv -- terrain elevation file
*.imp -- terrain ground cover file  

The NOISEMAP suite of programs can be downloaded from http://www.wasmerconsulting.com/baseops.htm
